# 🛒 ShopEase - Modern Product Catalog & Cart System

A professional, fully responsive e-commerce interface built with **Vanilla JavaScript**. This project demonstrates a clean architecture, dynamic data rendering, and a robust shopping cart system.

## 🌟 Features Overview

### 1️⃣ Task 1: Product Listing & Navigation
- **Semantic Structure:** Built using proper HTML5 tags (`<aside>`, `<main>`, `<article>`) for SEO and accessibility.
- **Dynamic Filtering:** Instant filtering by categories (Sports, Accessories, Electronics, Clothing).
- **Sorting Engine:** Sort products by price (Low to High / High to Low) to enhance user experience.
- **Modern Sidebar:** A sleek, persistent navigation menu inspired by premium minimalist stores.

### 2️⃣ Task 2: Shopping Cart & Checkout
- **Smart Cart Logic:** Add items to the cart with automatic quantity management (increments existing items).
- **Persistent Data:** Uses **Local Storage** to ensure your cart remains saved even after a page refresh.
- **Cart Management:** Full control to increase, decrease, or remove items directly from the cart drawer.
- **Checkout Experience:** - Real-time calculation of total quantities and prices.
    - **Custom Success Overlay:** A premium "Order Confirmed" screen (no boring browser alerts!).
    - Responsive "Slide-out" cart modal for a seamless mobile experience.
