# 📦 Dynamic Product Catalog

A professional, responsive product listing page built with **Vanilla JavaScript**. This project follows a modern sidebar layout for an intuitive user experience and high-quality design.

## 🚀 Features
- **Modern Sidebar UI:** Sleek category navigation (Sports, Accessories, Electronics, Clothing).
- **Dynamic Filtering:** Instant filtering logic based on selected categories.
- **Smart Sorting:** Sort items by price (Low to High / High to Low) or alphabetically.
- **Data Persistence:** Uses **Local Storage** to remember your filtering preferences.
- **Responsive Layout:** Works perfectly on mobile, tablet, and desktop.
- **Semantic Structure:** Built with `<aside>`, `<main>`, and `<article>` for optimal accessibility.
